# 球探比分——体育台球约教练

  "球探比分——体育台球约教练"app是一款线上预约台球教练的应用。
  
  主要功能:
  
  1、俱乐部:
    1.1: 提供了成都市热门的7家台球俱乐部;
    1.2: 每个俱乐部都配有3名该俱乐部下的教练;
    1.3: 每个教练的服务费均不同。
    
  2、教练:
    2.1: 教练和俱乐部分组，直观看到该俱乐部下的各个教练。
    
  3、我的:
    3.1: 我的打卡和我的点赞;
    3.2: 我的预约(已预约、已完成、已取消、全部预约)等预约;
    3.3: 我的收藏、联系客服、设置相关。
